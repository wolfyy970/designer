# Design quality skill

Use when shaping static pages for clarity and craft (no build step).

## Checklist

- Establish a clear typographic scale (heading vs body vs caption).
- Use consistent spacing steps (e.g. 4/8/12/16/24px rhythm).
- Keep primary actions visually dominant; secondary actions quieter.
- Prefer real content tone over placeholder gibberish.
- Ensure the focal hypothesis is obvious above the fold when relevant.

## Patterns

- **Above the fold:** headline + supporting line + primary CTA within first viewport.
- **Scanability:** short paragraphs, meaningful subheads, bullet lists for tasks.
- **Density:** mobile-first; avoid cramming more than one primary task per screen.
